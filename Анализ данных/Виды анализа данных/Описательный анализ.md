Это самый простой и распространенный тип аналитики, который используют при анализе данных. Используется для визуализации ключевых показателей, которые позволяют отслеживать их тенденции, также помогает понять, что происходит с данными в конкретный момент. **Обычно описательный анализ включает в себя [[Базовые методы агрегации данных|агрегирование данных]], их первичный анализ и визуализацию.**

> **Агрегирования данных** - это сбор, сортировка и форматирование данных для упрощения их дальнейшего анализа

**Первичный анализ включает в себя:**
- Определение [[Типизация данных#Базовые классификации данных|типа данных]]
- Определение пределов изменений типов данных
- Выявление ошибок в данных
- Выявление качественного поведения данных во времени и пространстве
- Выявление качественных и количественных соотношений между различными частям данных

> Одним из способов представления результатов описательной аналитики данных является их визуализация

**Основные шаги описательного анализа:**
1. **Определение метрик моделей данных** - определить ключевые параметры (метрики), которые хотите изучать и которыми будете оценивать качество данных и качество моделей их описывающих. **Метрики делятся на два вида:**
	- Бизнес-метрики - это клиент-ориентированные показатели, с помощью которых клиенты измеряют эффективность модели их процесса для своих задач. **Примеры:** показатели издержки и прибыли.
	- Прокси-метрики - это математические показатели, позволяющие субъективно оценить улучшение или ухудшение бизнес-метрик
> ***Здесь и далее "прокси-метрики" будут заменены на "метрики". Имейте это ввиду***
2. **Определение и поиск необходимых данных** - поиск и извлечение данных, необходимых для получения (расчета) желаемых параметров (метрик). Состоит из поиска и анализа источников данных.
3. **Извлечение и подготовка данных (переменных)** - если данные поступают из нескольких источников, то извлечение, объединение и подготовка данных для анализа является трудоемким процессом. **Этот шаг может включать в себя:**
	- Очистку данных для устранения несоответствий и ошибок в данных из разных источников
	- Преобразование данных в формат, подходящий для инструментов анализа
4. **Анализ данных** - это шаг включает в себя применение основных математических операций или преобразований к одной или нескольким переменным, а также расчеты необходимых метрик по ним.
5. **Представление данных** - представление данных в аудио-визуальных формах, которые понятны клиенту. Используется для упрощения понимания особенностей изучаемых данных заинтересованным сторонам.
