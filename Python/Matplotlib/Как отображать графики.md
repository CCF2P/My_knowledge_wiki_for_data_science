Визуализация графиков, выводимых библиотекой **Matplotlib**, зависит от контекста, в котором используется

### Построение графиков в сценариях (скриптах)
Для отображения графиков в сценарии (скрипте) используется метод `plt.show()`. Она запускает цикл обработки событий, ищет активные в настоящий моменты объекты *Figure* и открывает одно или несколько интерактивных окон для отображения вашего графика (графиков).

Например:
```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linespace(0, 10, 100)

plt.plot(x, np.sin(x))
plt.plot(x, np.cos(x))

plt.show()
```

> Команда `plt.show()` выполняет за нас множество операций в процессе взаимодействия с интерактивным графическом движком системы, т.е. скрывает от нас детали реализации

> **Важное замечание:** метод `plt.show()` <u>следует использовать <i>только один раз</i> в течение сеанса работы с Python</u>, и чаще всего его можно встретить в самом конце сценария. *Выполнение метода `plt.show()` несколько раз в сценарии может привести к непредсказуемому поведению* в зависимости от графического движка.

### Построение графиков из командной оболочки Python
**Matplotlib** можно использовать в интерактивной оболочке IPython. Сначала ее надо перевести в режим *Matplotlib*, при помощи команды `%matplotlib`.

После этого любая команда `plt.plot()` будет открывать окно графика с возможностью его изменения вызовом дальнейших команд.
> Некоторые изменение (например, модификация свойств уже нарисованных линий) не будут применяться автоматически, поэтому потребуется выполнить команду `plt.draw()`.

> Выполнять команду `plt.show()` в оболочке IPython в режиме **Matplotlib** не обязательно

### Построение графиков из блокнота Jupyter
Блокнот Jupyter - браузерный интерактивный инструмент для [[Введение|анализа данных]], позволяющий совмещать текстовое описание, код, график элемент HTML и многое другое в едином выполняемом документе.

Интерактивное построение графиков в блокноте Jupyter возможно с помощью команды `%matplotlib` и осуществляется так же, как в командной оболочек IPython.<br>
В Jupyter также есть две альтернативные возможности включения графиков непосредственно в блокнот:
- `%matplotlib inline` - включить в блокнот *статические* изображения графиков
- `%matplotlib notebook` - включить в блокнот *интерактивные* графики

> В Jupyter установлен режим по умолчанию - отображение *статических графиков*
