Сохранение выполняется с помощью команды `savefig`, например, можно сохранить изображение в файле PNG:
```python
fig.savefig('my_figure.png')
```

Это команда создаст в текущем рабочем каталоге файл с именем `my_figure.png`

Чтобы отобразить изображение, используется объект `Image` оболочки IPython:
```python
from IPython.display import Image
Image('my_figure.png')
```

Команда `savefig` определяет формат файла, исходя из расширения имени файла. В зависимости от установленного графического движка могут поддерживаться различные форматы файлов.<br>
Вывести список поддерживаемых форматов файлов можно с помощью следующего метода объекта `canvas` рисунка:
```python
fig.canvas.get_supported_filetypes()
```
