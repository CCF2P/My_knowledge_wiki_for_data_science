# Что такое Pandas
**Pandas** - библиотека на языке *Python* для анализа и обработки данных.

**Pandas** развивается как проект с открытым исходным кодом, который можно найти в репозитории на github (http://github.com/pandas-dev/pandas/).

В итоге можно выделить следующие преимущества **Pandas**:
- **Эффективная обработка данных:** **Pandas** предоставляет структуры данных, которые оптимизированы для обработки больших данных:
	- **Series** - одномерный массив с метками, который может содержать данные любого типа (*integer, string, объекты Python и т.д.*)
	- **DataFrame** - двумерная структура данных, которая хранит данные в виде двумерного массива или таблицы со строками и столбцами
- **Широкая поддержка форматов данных** - **Pandas** поддерживает следующие данные: CSV, Excel, базы данных SQL и JSON
- **Расширенные возможности манипулирования данными** - **Pandas** предлагает набор функций и методов для управления, преобразования и очистки данных
- **Анализ временных рядов** - **Pandas** имеет функции для работы с временными рядами, предоставляя поддержку индексирования по времени, пере дискретизацию, и операции сдвига во времени
- **Интеграция с экосистемой Python** - **Pandas** <u>легко интегрировать с другими популярными библиотеками в экосистеме *Python*</u>, такими как *NumPy, Matplotlib, scikit-learn*, что позволяет пользователям комбинировать сильные стороны различных библиотек в более комплексных сценариях.

# Установка Pandas
**Pandas** поддерживает следующие версии Python: <u>3.9, 3.10, 3.11, 3.12</u>

Команда для установки **Pandas** с помощью пакетного менеджера **PyPl**
```bash
pip install pandas
```
