**NumPy** - *Numerical Python* - библиотека, реализующая интерфейс для эффективного хранения и обработки данных.

Чтобы установить библиотеку используется команда:
```bash
pip install numpy
```

Чаще всего при подключении **NumPy** используется псевдоним *np*, т.е.:
```python
import numpy as np
```
