## 🚀 Как Начать

### Для Чтения
1. **В браузере:** Просто перейдите по [возможно появится ссылка на онлайн версию].
2. **Локально (рекомендуется):**
    - Установите [Obsidian](https://obsidian.md/) (бесплатен для личного использования).
    - Склонируйте этот репозиторий:
      ```bash
      git clone https://github.com/CCF2P/My_knowledge_wiki.git
      ```
    - Откройте папку `My_knowledge_wiki` как хранилище в Obsidian.
    - Начните с карты связей (**Graph View**) или с интересующей вас папки .

### Для Вклада и Синхронизации
1. Сделайте форк (Fork) этого репозитория.
2. Клонируйте свой форк на компьютер.
3. Откройте его в Obsidian, внесите изменения.
4. Закоммитьте и отправьте пул-реквест (Pull Request) с описанием изменений.
