При задании логического условия в предложении <b>WHERE</b> могут быть использованы операторы:
- <b>IN</b> - предикат (условие) с <b>IN</b> считается истинным, если значение поля, имя которого указано слева от <b>IN</b>, в точности *совпадает* с одним из значений, перечисленных в списке, указанном в скобках справа от **IN**. Пример:
```sql
SELECT *
FROM EXAM_MARKS
WHERE MARK IN (4, 5);
```
